Gewichtungen

Die Gewichtung orientiert sich an der Fibonacci-Folge.

Die Gewichtung pro Vergleichsparameter kann die Werte 0, 1, 2, 3, 5, 8 oder 21 annehmen.

Ist eine Aussage positiv formuliert, sind die Gewichtungen folgendermaßen aufgeteilt:
Stimme voll dagegen = 1, Stimme eher dagegen = 2, Ist mir egal = 3, Stimme eher zu = 5, Stimme voll zu = 8.

Ist eine Aussage negativ formuliert, sind die Gewichtungen folgendermaßen aufgeteilt:
Stimme voll dagegen = 8, Stimme eher dagegen = 5, Ist mir egal = 3, Stimme eher zu = 2, Stimme voll zu = 1.

Jeder Nutzer kann 3 Mal die besondere Gewichtung auswählen oder eine Frage überspringen:
Überspringen = 0, Ja / Nein, ist mir besonders wichtig = 21.
